## Project title - To-Do List Program (Python)

## Description
This is a simple Python program that allows users to add tasks (like "Finish Python assignment") to a list and view them.

It demonstrates basic Python concepts such as:
- Lists
- User input
- Loops

## How to Run

1. Make sure Python is installed on your system.

2. Open terminal or command prompt.

3. Navigate to the project folder:
   cd <DECODELABS>

4. Run the program.
  


## Project title - Expense Tracker (Python)

## Description

A simple Python command-line program that allows users to enter expenses one by one and calculates the total amount spent. It handles invalid inputs and ensures only numeric values are added.

## How to Run

1. Make sure Python is installed on your system.

2. Open terminal or command prompt.

3. Navigate to the project folder:
   cd <DECODELABS>
